# Architecture Decision Records (ADRs)

This folder contains the decision log for this service. Use ADRs to document significant architectural choices, including those that were rejected.

## How to add a decision
1. Copy `0000-template.md` to a new file named `NNNN-short-title.md` (e.g., `0001-choose-postgres.md`).
2. Fill in the context, decision, and consequences.
3. Submit a PR for review.
